libcronkite(3)
==============

NAME
----
libcronkite - a library used to fetch and preparse aur/aurjson data.


SYNOPSIS
--------
The archlinux aur.rpc mechanism requires url creation, json data fetching over
http, json data parsing, and eventual presentation to the end user.

The libcronkite library helps by doing all but the last step.


INTERFACES
----------
The libcronkite library provides the following interfaces:

*cronkite_get*::
    The query/poll interface.

*cronkite_cleanup*::
    The cleanup inerface.
