cronkite(1)
===========

NAME
----
cronkite - aurjson helper utility


SYNOPSIS
--------
*cronkite* ['OPTIONS'] ['OPTION-ARGUMENTS']


DESCRIPTION
-----------
The cronkite(1) command is a helper utility, for those writing AUR command line
utilities. The goal is to ease the usage of the aurjson interface, by providing
a pre-parsed output for more friendly grep'ing and sed'ing.

cronkite is powered by libcronkite. See libcronkite(3) for more details.


OPTIONS
-------
*-msearch* 'MAINTAINER-NAME'::
    Search for a package by 'MAINTAINER-NAME'.

*-search* 'SEARCH-TERM'::
    Search for a package by 'SEARCH-TERM'.

*-info* 'PKG-NAME'::
    Get info for a package named 'PKG-NAME'.

*-version*::
    Show version and exit.

*-help*::
    Show simple help and exit.


ENVIRONMENT
-----------
The following environment variables affect the execution of *cronkite*:

'CRONKITE_DELIMITER'::
    Specifies the delimiter/separator between the elements in the
    *OUTPUT FORMAT*. Defaults to `\t` if not set.

'CRONKITE_AURURL'::
    Specifies the destination url for queries. If not provided, the
    default AUR URL is used.


OUTPUT FORMAT
-------------
Each result in the output is emitted as a 'CRONKITE_DELIMITER' separated list,
containing the following aurjson result elements, in this order:

    Name
    ID
    Version
    CategoryID
    Description
    URL
    URLPath
    License
    NumVotes
    OutOfDate


EXIT STATUS
-----------
*0*::
    Success

*1*::
    Failure

*2*::
    No results found, or results unparsable.
